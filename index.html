About Work

This work consists of queries to a database using MySQL, which is a system based on queries and stands for structure query language. Throughout this page you will be seeing certain categories such as normal queries, updates to the database and queries. to delete information from the database.

I hope it can serve as help and guide to some queries and solutions to some specific problems.

Queries

1. Selecciona todos los usuarios cuyo nombre contiene la letra ‘a’.

Esta consulta selecciona todos los registros de la tabla users donde el campo names contiene la letra 'a' o 'A'.

2. Selecciona todos los usuarios cuyo apellido empieza con ‘S’.
